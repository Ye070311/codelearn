# CodeLearn v0.1 - 部署指南

## 第一步：上传到 GitHub

1. 打开 https://github.com 注册/登录账号
2. 点右上角 "+" → "New repository"
3. Repository name: 输入 codelearn
4. 设为 Public（公开，免费计划需要）
5. 点 "Create repository"

6. 在创建后的页面，复制这三行命令：
   （在下面的 "…or push an existing repository from the command line" 区域）
   然后打开你的电脑终端（cmd 或 PowerShell）执行

   或者直接下载安装 Git：https://git-scm.com/download/win
   安装后执行：
   cd C:\Users\admin\Documents\软件\code-learning-app
   git init
   git add .
   git commit -m "v0.1 初始版本"
   git branch -M main
   git remote add origin https://github.com/你的用户名/codelearn.git
   git push -u origin main

## 第二步：部署到 Render

1. 打开 https://dashboard.render.com 用 GitHub 登录
2. 点 "New +" → "Web Service"
3. 选 "Connect a repository" → 选 codelearn
4. 填写：

   Name: codelearn
   Region: Singapore
   Branch: main
   Runtime: Node
   Build Command: cd server && npm install && npx tsc
   Start Command: cd server && node dist/index.js
   Plan: Free

5. 点 "Create Web Service"
6. 等待 3-5 分钟，状态变成 "Live" 就成功了！
7. 访问 https://codelearn.onrender.com

## 后续更新代码

修改代码后，在项目目录执行：
git add .
git commit -m "更新说明"
git push

Render 会自动重新部署！

## 可选：配置自定义域名

在 Render 的 Settings → Custom Domain 里添加域名
